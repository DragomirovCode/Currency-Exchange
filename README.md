### Проект "Обмен валют"
Проект "Обмен валют" представляет собой REST API для описания валют и обменных курсов. Этот API позволяет просматривать и редактировать списки валют и обменных курсов, а также совершать расчёт конвертации произвольных сумм из одной валюты в другую.

### Мотивация проекта
Этот проект разработан с целью:

Изучения архитектурного шаблона MVC и его применения в разработке приложений.
Опыта работы с REST API, включая правильное именование ресурсов и использование стандартных HTTP кодов ответа.
Получения базовых навыков работы с SQL для создания и управления базой данных, в данном случае используется SQLite.

### Использование
Для начала работы с проектом необходимо склонировать его репозиторий и установить все зависимости. Далее следует запустить приложение на сервере и использовать API для взаимодействия с базой данных обменных курсов и валют.
